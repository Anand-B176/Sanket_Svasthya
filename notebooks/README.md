# Notebook Descriptions

## Overview

Jupyter notebooks for exploration and experimentation.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `01_data_exploration.ipynb` | Dataset analysis and visualization |
| `02_model_experiments.ipynb` | Model architecture experiments |
| `03_results_visualization.ipynb` | Results and metrics visualization |

## Usage

```bash
# Start Jupyter
jupyter notebook

# Or use JupyterLab
jupyter lab
```

## Note

Notebooks are optional and used for exploratory work.
Core functionality is in Python scripts.
